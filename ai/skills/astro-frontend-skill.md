# Astro Frontend Skill

## Use When

Use this project skill when adding or changing Astro pages, layouts, components,
or data files for the Travel Agent frontend.

## Workflow

1. Confirm the active branch is `develop`.
2. Read `AGENTS.md`.
3. Keep the work scoped to the requested step.
4. Prefer Astro components for static UI.
5. Use React only for interactive islands.
6. Run build and Astro check before reporting.
