# Travel UIUX Skill

## Use When

Use this project skill for taxi, tour, travel, package, gallery, CTA, and mobile
booking UI work.

## Principles

- Prioritize mobile travelers.
- Make WhatsApp CTA access obvious.
- Use premium but restrained visual styling.
- Let real images carry travel atmosphere when assets are available.
- Keep cards scannable for route, duration, capacity, and price data.
