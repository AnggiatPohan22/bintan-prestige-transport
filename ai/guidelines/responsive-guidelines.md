# Responsive Guidelines

## Breakpoints

- Mobile: 360px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+
- Large desktop: 1280px+

## Requirements

- Start with one-column mobile layouts.
- Use two-column grids from tablet where content benefits from it.
- Use three-column grids only when cards remain readable.
- Keep CTA buttons at least 44px tall.
- Preserve image aspect ratios.
- Avoid horizontal overflow.
- Keep mobile spacing tighter than desktop spacing.
