# SEO Guidelines

## Foundation

- Every page must use `BaseLayout`.
- `BaseLayout` must include `SEO.astro`.
- Page-level title and description should be specific.
- Canonical and OG image should be easy to pass per page.

## Future Enhancements

Add these only when a later step asks for them:

- LocalBusiness schema
- TaxiService schema
- TravelAgency schema
- Service schema
- FAQ schema
- Breadcrumb schema
- Sitemap
- Robots
